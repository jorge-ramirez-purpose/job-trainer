# Interview Prep App

A React + TypeScript + Vite interview preparation application for frontend developers.

## Features

- **Quiz Mode**: Interactive questions with multiple choice answers and explanations
- **Flashcard Mode**: Study questions in flashcard format
- **Progress Tracking**: localStorage-based progress tracking across sessions
- **Topic Coverage**: JS closures, useState batching, useEffect dependencies, and stale closures

## Topics Covered

- **React**: useState batching, useEffect dependencies, stale closures
- **JavaScript**: Closures and scope

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── types/          # TypeScript interfaces
├── data/           # Question data
└── components/     # React components
```
